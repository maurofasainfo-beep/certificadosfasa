# Variáveis de Ambiente

Obrigatorias:

- `NEXT_PUBLIC_SUPABASE_URL`
- `NEXT_PUBLIC_SUPABASE_ANON_KEY`
- `SUPABASE_SERVICE_ROLE_KEY`
- `CERT_ENCRYPTION_KEY`
- `CRON_SECRET`
- `EUATENDO_API_URL`
- `EUATENDO_API_TOKEN`
- `EUATENDO_INSTANCE_ID`
- `EUATENDO_PROVIDER_ENABLED`

Nunca exponha `SUPABASE_SERVICE_ROLE_KEY`, `CERT_ENCRYPTION_KEY`, `CRON_SECRET` ou `EUATENDO_API_TOKEN` no frontend.
