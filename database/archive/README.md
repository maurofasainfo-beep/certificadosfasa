# Archive SQL

SQLs antigos, substituidos, correcoes avulsas e scripts historicos.

Estes arquivos foram preservados para auditoria e rastreabilidade. Eles nao sao a fonte operacional atual.

Quando houver conflito:

1. Codigo atual prevalece.
2. `database/schema/supabase_schema.sql` prevalece para banco novo.
3. `database/migrations/` prevalece para evolucao incremental.
4. `docs/SYSTEM_CONTEXT.md` prevalece para contexto oficial.
